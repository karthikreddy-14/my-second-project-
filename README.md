# String Cleaner Tool

## 📌 Description
This is a Java-based tool that cleans a given string by removing spaces and special characters.

## 🚀 Features
- Removes spaces from input string
- Removes special characters
- Keeps only letters and numbers

## 💻 Tech Used
- Java
- Regular Expressions

## ▶️ How to Run
1. Compile the program:
   javac StringCleaner.java
2. Run the program:
   java StringCleaner

## 📌 Example
Input: g eeks@for#ge eeks!!  
Output: geeksforgeeks
